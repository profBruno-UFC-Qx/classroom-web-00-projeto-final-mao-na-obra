# Telas implementadas

- [Tela de Inicio](./tela-inicial)
  - Na tela inicial, os usuários podem visualizar informações sobre a plataforma, conhecer os principais serviços disponíveis e acessar rapidamente as opções de login e cadastro.

- [Tela de Login](./tela-login/index.html)
  - A tela de login é responsável pela autenticação dos usuários cadastrados. Por meio dela, os usuários podem acessar suas contas utilizando suas credenciais de acesso de forma segura.

- [Tela de Cadastro](./tela-cadastro/index.html)
  - A tela de cadastro permite que novos usuários criem suas contas na plataforma. Durante o processo, o usuário pode informar seus dados pessoais e criar seu acesso ao sistema.

- [Tela do Dashboard](./dashboard/dashboard.html) 
  - A dashboard centraliza as principais funcionalidades e informações da plataforma em um único ambiente. Visualizar informações importantes:
    - Acompanhar solicitações;
    - Gerenciar serviços;
    - Navegar entre as funcionalidades do sistema;
    - Acessar configurações da conta.
